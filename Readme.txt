# OmniGrejas Admin

OmniGrejas Admin é o painel de administração da plataforma **OmniGrejas**, uma rede social voltada para a comunidade religiosa. Este painel permite a gestão de igrejas, usuários, categorias e funcionalidades essenciais da aplicação.

## Tecnologias Utilizadas
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** PHP (Laravel futuramente)
- **Banco de Dados:** MySQL
- **Autenticação:** JWT (futuramente)

## Funcionalidades Principais
- **Gerenciamento de Igrejas:** Cadastro, edição e remoção de igrejas.
- **Controle de Usuários:** Visualização, edição e remoção de usuários.
- **Categorias:** Administração das categorias dentro da plataforma.
- **Mensagens e Contato:** Envio de mensagens para usuários diretamente pelo painel.
- **Segurança:** Controle de acesso a funcionalidades administrativas.
